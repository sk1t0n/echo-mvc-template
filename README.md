# echo-mvc-template

This is a project template for Echo MVC apps. Tech stack: [Echo](https://echo.labstack.com/), [Scriggo](https://github.com/open2b/scriggo).

## Install

```sh
go mod download
npm i -D
```

## Minify CSS and JS

```sh
npm run build
```

## Start

```sh
go run cmd/web/main.go
```
