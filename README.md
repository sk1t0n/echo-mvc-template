# fiber-mvc-template

This is a project template for Fiber MVC apps. Tech stack: [Fiber](https://github.com/gofiber/fiber), [Scriggo](https://github.com/open2b/scriggo).

## Install

```sh
go mod download
npm i -D
```

## Minify CSS and JS

```sh
npm run build
```

## Start

```sh
go run cmd/web/main.go
```
